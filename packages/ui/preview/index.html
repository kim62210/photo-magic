06 · Responsive

320px부터 4K까지, 동일한 캐릭터.

XS · 0px+

단일 컬럼, 캔버스 100% 너비, 하단 고정 툴바 64px. 디스플레이 폰트 -1단계 (88 → 64). 페이지 패딩 16px.

SM · 640px

2컬럼 그리드 가능. 카드 가로 배치. 페이지 패딩 24px.

MD · 768px

8컬럼 그리드. 페이지 패딩 32px. 사이드바 등장 시작 (편집기 좌측).

LG · 1024px

12컬럼 그리드. 편집기 3컬럼 레이아웃 (도구 240 + 캔버스 + 속성 280). 페이지 패딩 48px.

XL · 1280px

컨테이너 max 1200px. 디스플레이 폰트 +0.5단계. 4K 프리뷰 가능.
